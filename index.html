HTML Basic Semantic Tags for Content Structure

Tag Definition Features & Sidenotes
Header Top part of a site, used for basic intro information, such as main Heading, Logo etc
Nav Used to navigation part of a site. Mainly includes hyperlinks, as internal so external
Main Main part of a site, which consict main content
Footer Most bottom part of a side, mostly used for "tech" information such as contacts, additional links etc
Article Consist main usefull information for viewers. May include any type of content: from text to media
Aside Commonly consist additional to main Article info
H1, H2, H3... Headings for all type of content. Usefull for SEO
